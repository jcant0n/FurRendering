# FurRendering
Test fur rendering shells technique, rendering using instanced and alphatest.

#### Testing Parameters: 
density = 0.4
minHairLength = 0.5
numLayers = 50
shartShadowValue = 0.2
MaxHairLength = 0.2
Texture Size = 1024
